# Hi, I'm Daniel

## What I'm Building

**Agentic AI Developer Workflows**
Claude Code 기반 커스텀 Skill과 sub-agent를 직접 설계해, 스키마 설계·API 스캐폴딩·마이그레이션 작성 같은 다단계 엔지니어링 작업을 리뷰 가능한 자동화 파이프라인으로 전환. GitHub Actions에 AI 기반 PR 리뷰 게이트까지 도입.
`Claude Code` `MCP` `GitHub Actions`

**Frontend Architecture & Design Systems**
대규모 React 애플리케이션에서 공통 컴포넌트(버튼·인풋·바텀시트 등)와 재사용 가능한 Flow 패턴을 설계하고, Drawer 기반 UI를 라우터 기반 구조로 전환하는 등 확장 가능한 네비게이션·상태 아키텍처를 정립.
`React` `MUI` `React Router`

**Backend Scenario Test System**
API 레이어에 시나리오 기반 테스트 컨벤션을 직접 정립하고, 정책 변경 시 관련 시나리오를 함께 갱신하는 워크플로우를 구축. 이 컨벤션을 Claude Code 사내 Skill(`scenario-test`)로 자동화해 팀 전체에 배포.
`Jest` `Node.js` `Claude Code`

**Fullstack Reliability & Multi-Region Platform**
Node.js/MySQL 백엔드에서 PII 스크러빙, 동시성 안전 트랜잭션, 데드락 방지 로직을 직접 구현하고, 다국가 서비스 확장을 위한 자동 번역 파이프라인과 국가별 권한 스코핑을 설계.
`Node.js` `MySQL` `Sentry`
